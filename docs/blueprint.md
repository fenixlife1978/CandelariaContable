# **App Name**: LoanStar Accounting

## Core Features:

- Income Tracking: Record all income from loans with date, amount, and description.
- Expense Tracking: Record all expenses related to loans with date, amount, and description.
- Capital Calculation: Automatically calculate capital based on income and expenses.
- Accounting Columns Display: Display income, expenses, and capital in a columnar accounting format.
- Loan Summary: Generates AI-driven monthly summaries with tool that suggests ways to improve cash flow.  The AI model uses its tool to include details and suggest actions based on financial benchmarks. The AI can reason on when and when not to offer recommendations based on historical data and user settings. 

## Style Guidelines:

- Primary color: Dark green (#00704A), reminiscent of Starbucks' brand, for trustworthiness.
- Background color: Very light green (#F0FAF5), a desaturated tint of the primary for a clean backdrop.
- Accent color: Gold (#D4AC0D) for highlights and important actions, adding a touch of sophistication.
- Body and headline font: 'PT Sans', a humanist sans-serif for both headlines and body.
- Use simple, line-based icons for income, expenses, and capital to maintain a clean look.
- Responsive layout that adapts to different screen sizes, ensuring usability on all devices.
- Subtle animations for transitions and loading states to enhance the user experience.
